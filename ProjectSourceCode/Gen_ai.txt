 You’re super close — the issue is just a few small mistakes that break the offcanvas.

### ❌ Problems in your code

1. **Typo in `data-bs-toggle`**

```html
data-bs-toggle="offfcanvas"
```

👉 should be:

```html
data-bs-toggle="offcanvas"
```

---

2. **Missing closing `</button>`**
   Your button never closes, which breaks the DOM.

---

3. **You have TWO `<body>` tags**
   That can mess up Bootstrap JS behavior.

---

4. **Navbar class is wrong**

```html
navbar-black bg-black
```

👉 Bootstrap doesn’t have `navbar-black`. Use:

```html
navbar-dark bg-dark
```

---

5. **Duplicate Bootstrap JS**
   You included it twice — not fatal, but messy.

---
